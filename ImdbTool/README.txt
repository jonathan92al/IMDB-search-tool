IMDB MOVIES WEB SCRAPER

CREATED BY:   YEHONATAN ARBEL

RUNNING TOOL:

1) Extract ImdbTool folder
2) dist > Tool > Tool(.exe)
3) Type a string and press Enter - text file named after your string (MyString, for example) will be created in dist folder (may take a few minutes!)

IMPORTANT NOTES:

* Not all exceptions were handled
* MPAA rating feature is missing
